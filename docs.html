Quick Start Guide

Step 1: Install the SDK

pip install gpuaas

Step 2: Get Your API Key

Navigate to your dashboard at https://console.gpuaas.com and generate an API key from Settings → API Keys.

💡 Security Best Practice: Store your API key in environment variables, never commit to version control.

Step 3: Launch Your First Instance

from gpuaas import Client

# Initialize client
client = Client(api_key="your-api-key")

# IaaS: Launch bare-metal GPU instance
instance = client.instances.create(
    gpu_type="A100-80GB",
    image="ubuntu-22.04-cuda-12.2",
    region="us-east-1"
)

print(f"Instance created: {instance.id}")
print(f"SSH access: ssh root@{instance.ip_address}")

IaaS - Infrastructure as a Service

Full control over GPU infrastructure with direct hardware access. Ideal for research, custom deployments, and HPC workloads.

Available GPU Models

  • H200 SXM: 141GB HBM3e - Large-scale LLM training
  • H100 SXM: 80GB HBM3 - Production LLM training
  • A100 SXM: 80GB HBM2e - Versatile training & inference
  • L40S: 48GB GDDR6 - Optimized inference

Create IaaS Instance

# Full IaaS configuration
instance = client.instances.create(
    gpu_type="H100-80GB",
    gpu_count=8,  # Multi-GPU node
    cpu_cores=128,
    ram_gb=1024,
    storage_type="nvme",
    storage_gb=10000,
    network_speed="100gbps",
    image="ubuntu-22.04-cuda-12.2",
    ssh_keys=["ssh-rsa AAAAB3..."],
    region="us-east-1",
    vpc_id="vpc-12345"
)

PaaS - Platform as a Service

Managed machine learning platforms with pre-configured environments. Focus on model development while we handle infrastructure.

Deploy Managed Jupyter Environment

# Launch managed Jupyter with PyTorch
notebook = client.notebooks.create(
    name="my-ml-workspace",
    gpu_type="A100-40GB",
    framework="pytorch-2.1-cuda-12.1",
    storage_gb=500,
    auto_shutdown_minutes=60,
    shared_users=["user2@example.com"]
)

print(f"Access notebook: {notebook.url}")
print(f"Token: {notebook.access_token}")

Supported Frameworks

  • PyTorch 2.1+ with CUDA 12.x
  • TensorFlow 2.14+ with GPU support
  • JAX with XLA acceleration
  • Keras 3.0+ with multi-backend
  • Custom frameworks via Docker

SaaS - Software as a Service

Ready-to-use AI APIs with no infrastructure management. Perfect for production applications requiring immediate deployment.

Text Generation API

import requests

# Generate text using SaaS API
response = requests.post(
    "https://api.gpuaas.com/v1/text/generate",
    headers={
        "Authorization": "Bearer your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "llama-3-70b",
        "prompt": "Explain quantum computing",
        "max_tokens": 1000,
        "temperature": 0.7,
        "top_p": 0.9
    }
)

result = response.json()
print(result["generated_text"])

Available SaaS APIs

  • Text Generation: LLaMA, Mistral, GPT models
  • Image Generation: Stable Diffusion, DALL-E class
  • Computer Vision: Object detection, segmentation
  • Speech: Whisper for transcription
  • Embeddings: Text and image embeddings
