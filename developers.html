Quick Start - IaaS API

Launch GPU instances programmatically using our Infrastructure API

pip install ai-factory-sdk

from ai_factory import GPUClient

# Initialize client
client = GPUClient(api_key="your_api_key")

# Launch GPU instance
instance = client.create_instance(
    gpu_type="NVIDIA-H100",
    gpu_count=2,
    memory_gb=128,
    region="us-east-1"
)

print(f"Instance ID: {instance.id}")
print(f"Status: {instance.status}")
print(f"IP Address: {instance.ip_address}")

PaaS - Model Training Example

Train AI models using our managed platform

from ai_factory import TrainingPlatform

# Initialize training platform
platform = TrainingPlatform(api_key="your_api_key")

# Create training job
job = platform.create_training_job(
    name="bert-fine-tuning",
    framework="pytorch",
    gpu_type="A100",
    gpu_count=4,
    script="train.py",
    dataset="s3://my-bucket/dataset",
    hyperparameters={
        "learning_rate": 2e-5,
        "batch_size": 32,
        "epochs": 3
    }
)

# Monitor training
for log in job.stream_logs():
    print(log)

SaaS - Inference API

Call pre-trained models via REST API

import requests

API_URL = "https://api.ai-factory.com/v1/inference"
API_KEY = "your_api_key"

# Text generation
response = requests.post(
    f"{API_URL}/text-generation",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={
        "model": "gpt-4",
        "prompt": "Explain quantum computing",
        "max_tokens": 500,
        "temperature": 0.7
    }
)

result = response.json()
print(result["generated_text"])
